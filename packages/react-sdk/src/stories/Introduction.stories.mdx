# Airbolt React SDK Interactive Demo

Welcome to the **Airbolt React SDK** interactive demo! This is your playground for exploring and understanding how to integrate AI chat capabilities into your React applications.

## üöÄ Quick Start

Airbolt provides two main ways to add AI chat to your React app:

### 1. ChatWidget Component

A pre-built, fully-featured chat interface that works out of the box:

```jsx
import { ChatWidget } from '@airbolt/react-sdk';

function App() {
  return <ChatWidget baseURL="https://your-api.com" />;
}
```

### 2. useChat Hook

Build your own custom chat UI with complete control:

```jsx
import { useChat } from '@airbolt/react-sdk';

function CustomChat() {
  const { messages, input, setInput, send, isStreaming } = useChat({
    baseURL: 'https://your-api.com',
    // streaming: true is the default
  });

  // Build your own UI...
}
```

## üîß Key Features

### üåä Streaming Responses (New Default!)

Experience real-time AI responses that stream word-by-word:

- **Better UX** - Users see responses immediately as they're generated
- **Default behavior** - Streaming is enabled by default for both ChatWidget and useChat
- **Easy control** - Disable with `streaming={false}` when needed
- **onChunk callback** - Handle individual chunks for custom behaviors

```jsx
// Streaming is the default
<ChatWidget baseURL="https://your-api.com" />

// Or explicitly control it
<ChatWidget
  baseURL="https://your-api.com"
  streaming={false}  // Use chatSync for non-streaming
/>

// With useChat hook
const { messages, isStreaming } = useChat({
  baseURL: 'https://your-api.com',
  onChunk: (chunk) => console.log('Received:', chunk)
});
```
